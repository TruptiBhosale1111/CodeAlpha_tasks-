-Simple Rule-based Chatbot-
This project is a basic chatbot developed using Python.
It interacts with the user by taking input and giving responses based on predefined rules. 
The chatbot can understand simple messages like greetings and replies accordingly.

-About Project-
This chatbot works on keyword matching. 
It checks the user input and responds using conditions like if-elif. 
It continues the conversation until the user types "bye" to exit.

-Purpose of project-
1.How chatbots work at a basic level
2.Use of loops and conditional statements in Python
3.How to take input from users and respond

-How it works-
->The program asks the user for their name
->It continuously takes input from the user
->It matches keywords like "hello", "how are you", and "bye"
->Based on the input, it gives a fixed response
->The loop stops when the user enters "bye"
